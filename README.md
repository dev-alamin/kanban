# Kanban Board - React + Vite + Tailwind CSS

A modern kanban board application built with React, Vite, and Tailwind CSS for fast development and styling.

## Tech Stack

- **React** - UI library for building interactive components
- **Vite** - Next-generation frontend build tool with HMR (Hot Module Replacement)
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **ESLint** - Code quality and consistency enforcement
